Projeto de API Web backend para gerenciar a entidade Livro.

Tecnologias: TypeScript, Node.js, Express, TypeORM, SQLite

Endpoints:
- POST /api/livros - criar um novo livro
- GET /api/livros - listar todos
- GET /api/livros/:id - obter por id
- PUT /api/livros/:id - atualizar completamente
- PATCH /api/livros/:id - atualizar parcialmente
- DELETE /api/livros/:id - remover
